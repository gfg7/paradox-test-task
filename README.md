# paradox-test-task

С использованием StimulSoft реализовать генерацию отчета PDF из JSON-файла.
На .NET сделать приложение.
У приложения одна экранная форма. На ней сделать ввод JSON (указать путь к файлу или загрузить
файл) и кнопку «Создать отчет». Результат – отчет в формате PDF по заранее созданному шаблону
в StimulSoft
